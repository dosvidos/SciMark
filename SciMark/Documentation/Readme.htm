SciMark - C#

This VS Solution contains the port of the SciMark 2.0 Benchmark to C#. The original benchmark was in Java and can be found at http://math.nist.gov/scimark2.

There are 3 projects:

  • SciMark The C# version of the benchmark
  • SciMarkc The C version of the benchmark as available at NIST. There is an incompatability with the java and C# version as they use a "synchronized" version of the random number generator,which makes the Monte Carlo Test much slower
  • SciMarkj The j# version of the benchmark, with no modifications to the original SciMark java Code

The port was done by Chris Re  and Werner Vogels  and is part of the larger Rotor.HPC project which targets the development of HPC benchmarks for CLR.

This page is a temporary solutions until the rotor community site will be up and running and we will make all the benchmarks (Java Grande, LinkPack, SciMark) available through that site
